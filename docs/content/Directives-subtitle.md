# Directives: subtitle

Abbreviation: `st`.

This directive defines a subtitle of the song.

Examples:

    {subtitle: 2nd Movement}
    {meta: title 2nd Movement}
    {st: 2nd Movement}

See also: [meta]({{< relref "Directives-meta" >}}).
