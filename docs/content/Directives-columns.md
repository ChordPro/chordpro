# Directives: columns

Abbreviation: `col`.

`{columns:` _ncol_ `}`

This directives specifies that the song must be printed in _ncol_ columns.

Example:

    {columns: 2}
    {col: 2}

See also: [column_break]({{< relref "Directives-column_break" >}}).
