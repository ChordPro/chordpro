# Directives: chorus

This directive indicates that the song chorus must be played here. 

Examples:

    {chorus}
    {chorus: Final}

In the second form, the argument is used as a label for the chorus. 

See also: [start_of_chorus]({{< relref "Directives-env_chorus" >}}),
