Important notes for running ChordPro on macOS

This is an unsigned application for macOS 12 and later.

YOU CANNOT JUST OPEN THE APPLICATION

Due to security reasons, an unsigned application cannot just be opened.

Signing you application, even for distribution outside an Apple Store requires a yearly paid supscription.

This is not unfair, however, there are no members in the ChordPro community able to provide a signed binary at the moment.

For more information, see this Apple document.

How to open the application

  • `Right-click` the application and choose ‘open’
  • You will get a warning that the application is not verified and you are suggested to move int to the bin. Press `cancel`.
  • `Right-click` again and choose ‘open’ again.
  • Now you can choose ‘open anyway’ and the application should start.

Next time you can just open it without any warnings.

Even though the build is `Intel` only it will still run on arm-macs as well.

The reason it is not an universal binary is that an unsigned arm-version will simply refuse to open...
