ChordPro for macOS

What is ChordPro

ChordPro, the application, is first and foremost a graphical application and the reference implementation for the ChordPro standard:

“A simple text format for the notation of lyrics with chords”.

It has a very pleasant and easy to use interface that gives you the basics of using ChordPro; both the application as well its text-format.

Advanced usage

The package you downloaded also contains a very powerful but maybe hard-to-use application that you have to run in the Teminal.

Expert users can use this Teminal version for even more complex operations.

It is not just drag and drop

Providing applications for the Mac platform is not as easy as it seems. For the users, its mostly just drag and drop. It is not that easy for developers. Apple has a lot of security in place to make sure your drag and drop will be safe for you.

It needs a lot of time, effort and yeh... money for developers to provide such application for the Mac.

ChordPro is not a commercial product. It is an Open Source application made by volunteers. You are welcome to download the source, compile it and enjoy!

That is not realistic for must users, especially not on the Mac. We would love to provide you the expected experience but we can’t. There is nobody in the ChordPro community with an Apple Developers certificate to provide you this ready to go experience.

The Gatekeeper

Your download contains an almost ready to go ChordPro; except that the application is only ad-hoc signed by us and not by an approved Apple Developer with a valid certificate. So, the security system on your Mac will give an error when you double-click the application. You have met the Gatekeeper. Depending of the version of macOS you have a different message and have to do different additional handlings to open ChordPro.

  1. Drag the application to your Applications folder.
  2. Open the application by double-clicking on it.

A prompt saying '“ChordPro“ Not Opened' will appear.

The message will be more or less like this, depending on your macOS version:

Apple could not verify “ChordPro.app“ is free of malware that may harm your Mac or compromise your privacy.

There is a small 'question mark' button at the right top of the dialog that opens the 'Mac User Guide' with more information.

On macOS versions before 15 (Sequoia) it was common practice to right-click the application and choose Open from the menu. That trick will bypass the gatekeeper and let you open the application.

Unfortunately, that does not work anymore on Sequoia and you have to go trough System Settings:

  1. Go to System Settings -> Privacy & Security.
  2. Scroll down to find ChordPro, and an option to "Open Anyway".
  3. Choose "Open Anyway".
  4. Authenticate as an administrator.
  5. ChordPro will now open.

ChordPro is saved as an exception to your security settings and you can open it in the future by double-clicking it, just as you can with any other application.

The Terminal Application

The Terminal application, simply called chordpro is part of the package.

To use the Terminal application you first have give the GUI application permission to run on your system. Do that first.

Temporarily add it to your system path

To temporarily add the chordpro command line application to the system path issue the following command in the Terminal window:

export PATH="/Applications/ChordPro.app/Contents/Resources/cli":$PATH

Now you can use the chordpro command line application in the Terminal without its full path, however, when you close the Terminal you have add the path again the next time. It is recommended to add it permanently; see below.

Permanently add it to your system path

To add the command line application permanently to your system path there is no simple command for the Terminal. However, it is not difficult.

First, create a file pointing to your chordpro CLI directory in the Terminal:

echo "/Applications/ChordPro.app/Contents/Resources/cli" > ~/Desktop/chordpro

This will create a file with the location of the chordpro CLI command on your Desktop. You have to move this file with the Finder so the Terminal will see it:

  1. In the Finder, go to its menu at the top and 'Go -> Go to Folder...'
  2. In the dialog, enter `/etc/paths.d`. It should auto-complete and if you press the `Enter` key it will open a new Finder window.
  3. Now Drag & Drop' the 'chordpro' file you just created on your Desktop to this folder. It will ask for your administration password.
  4. Open a new Terminal window and check if it is all ok:
chordpro --about

This should show you information about chordpro if all went well.
